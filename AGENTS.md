# AGENTS.md

## 项目目标

本项目的目标是：在 MMDetection 3.x（当前以 mmdetection 3.3.0 为主要参考）框架下，重新实现一个基于 DINO 的半监督目标检测基线，用于后续复现和扩展 Semi-DETR 的核心思路。

当前阶段的首要任务不是一次性完整复刻 Semi-DETR 的全部细节，而是先基于官方半监督检测框架，使用 `SemiBaseDetector` 包裹一个 `DINO` 检测器，构建一个能够完整跑通“训练 -> 推理 -> 验证 -> 反向传播”全流程的最小可用半监督目标检测模型。

实现时应尽可能复用 MMDetection 3.x 官方已有能力，只在确有必要时新增、重写或覆盖相关组件。

---

## 项目背景

我当前从事的是半监督目标检测（Semi-Supervised Object Detection, SSOD）相关研究。

当前半监督目标检测方向中，可靠的 SOTA 参考之一是 Semi-DETR。但 Semi-DETR 的公开实现基于 MMDetection 2.x 体系，而本项目希望在 MMDetection 3.x 体系下重新实现其核心思路，并逐步迁移到新的代码结构和训练范式。

因此，本项目当前的工程目标是：

1. 基于 MMDetection 3.x 的官方半监督检测教程与实现方式，搭建新的工程骨架。
2. 先实现一个“`SemiBaseDetector` + `DINO`”的半监督目标检测基础模型。
3. 保证该模型能够正确完成：
   - 有监督分支前向；
   - 无监督分支前向；
   - teacher / student 协同；
   - loss 计算；
   - 训练反向传播；
   - 推理；
   - 验证评估。
4. 在这个基础上，再为后续进一步贴近 Semi-DETR 的设计预留扩展空间。

## 代码实现原则

### 1. 以“尽量复用官方实现”为第一原则
你写代码时，默认优先级如下：

1. 直接复用 MMDetection 3.x 官方现成组件；

- 易于改阈值、损失权重、数据路径；
